# Code of Conduct

PYXIS3 repositories are public engineering work. Discussion stays technical, professional, and on-topic.

## Standards

- Engage with the work, not the person
- No identity-based attacks
- Disagree without disrespect — strong opinions are welcome, condescension is not
- Take feedback seriously even when you disagree with it

## Reporting

Email [ops@pyxis3.ai](mailto:ops@pyxis3.ai) for any conduct issue. Reports are kept confidential.

## Scope

Applies in:
- Issue discussions, PR reviews, and code comments across all PYXIS3 repos
- Any community channel attached to PYXIS3 (Discord/Slack if/when created)
- In-person events organised by PYXIS3
